# 🚀 Personal Portfolio

Welcome to my personal portfolio website repository! This project showcases my skills, experience, and the various projects I have worked on.

## ✨ Features

- **Modern UI/UX**: Designed with a clean and professional look.
- **Responsive Design**: Fully responsive and optimized for desktop, tablet, and mobile devices.
- **Projects Showcase**: A dedicated section to display my best work.
- **About Me**: Information about my background and skills.
- **Contact**: Easy ways to reach out to me.

## 🛠️ Tech Stack

- **Frontend Framework**: [React.js](https://react.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## 🚀 Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your system.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Muhfaizr21/porto.git
   ```
2. **Navigate to the frontend directory:**
   ```bash
   cd porto/frontend
   ```
3. **Install the dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser to view the website.
